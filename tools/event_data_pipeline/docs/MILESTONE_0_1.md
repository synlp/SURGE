# Milestone 0.1：data9 到 Unified 中间层

日期：2026-07-13

## 已完成

- 独立 Python 工程与虚拟环境；
- `UnifiedPost`、`UnifiedInteraction` 标准对象；
- data9 ZIP 流式适配器；
- 主帖、回复、引用帖展开；
- UTC 时间、语言代码、互动数和 K/M/B 浏览量规范化；
- 稳定平台帖子 ID、正文哈希和来源溯源；
- 同事件平台 ID 去重及可选精确正文去重；
- 事件时间窗口裁剪；
- 标准帖子、互动边、重复记录和拒绝记录输出；
- 单事件质量报告和多事件批处理报告；
- Unified 输出一致性验证器；
- 按事件独立时间窗的配置批处理；
- 项目安装与命令行入口。
